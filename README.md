# Todoer
A simple Todo App

### You can:
- Create a 'Todoer' which acts as a list of todo items
- Create a todo item by providing a description for it
- Mark it as done
- Delete it
- Delete a 'Todoer'

Todoers and todo items are persistent across app restarts and device reboots.
